Markup project

Structure based on Html5Boilerplate.

Styles powered with Sass and Compass.

In /img/base64 there are images for IE8 and IE7. For all modern browsers they are converted to base64 images.

IE compatibility.
version 9 - full suport
version 8 - graceful degradation
version 7 - graceful degradation, minor bugs in structure.
version 6 - unsupported

IMPORTANT-------------->
Don't touch core.css!!!
Use custom.css for additional styles.
